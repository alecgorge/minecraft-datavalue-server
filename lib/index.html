Minecraft Data Value API

Get block and item names, data values, variations/subitems and images. For free. Data is pulled and cached from Minecraftdatavalues.com

Read the API docs free! I'm on GitHub: alecgorge/minecraft-datavalue-server!

API Endpoints

  • /blocks/ — Returns all available information about all blocks/items.

  • /blocks/images/ — Returns a list of all paths to all the images.

  • /blocks/names/ — Returns all known block/item names.

  • /blocks/ids/ — Returns all known block/item IDs.

  • /blocks/:id/ — Returns all available information for a particular block/item ID.

    • /blocks/:id/image/ — Returns a 16x16 PNG representation of the block/item.

    • /blocks/:id/:data_value/ — Returns all available information for a particular block/item with the given ID and with the given data value.

      • /blocks/:id/:data_value/ — Returns a 16x16 PNG representation of the block/item with the given ID and the given data value.

  • /images/:pic_name.png — Returns a png of the requested image.

Sample Reponse

All responses come with Etagheaders. Please pass that value from that Etagback in your next request as the If-None-Matchheader. If the request is still the same, the server will respond with a 304response with no body.

JSONP
